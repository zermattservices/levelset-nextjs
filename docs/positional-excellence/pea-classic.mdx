# Classic View

The Classic View shows positional ratings in a spreadsheet-style format. It's great for printing, displaying on screens, and quick visual comparisons.

## How to Access Classic View

**Option 1: From the Dashboard**
1. Go to **Positional Excellence**
2. Click **Go to Classic View**

**Option 2: Direct URL**
Navigate to `/positional-excellence-classic` in your browser.

## What You'll See

The Classic View displays a grid with:

- **Rows** = Team members
- **Columns** = The 5 rating criteria for the selected position
- **Cells** = Average rating for each criterion (color-coded)

## Using the Controls

### Select a Position

Use the dropdown at the top to choose which position to view. The grid updates to show ratings for that position.

### Toggle FOH/BOH

Use the toggle to switch between:
- **FOH** - Front of House positions only
- **BOH** - Back of House positions only

### View Options

- **Expand/collapse rows** to see more or less detail
- **Scroll horizontally** if there are many criteria columns

## Best Uses for Classic View

### Team Meetings

1. Display the Classic View on a screen
2. Walk through team performance by position
3. Recognize top performers
4. Identify areas for group training

### One-on-One Reviews

1. Open the Classic View filtered to one person
2. Print or share the screen
3. Discuss specific criteria scores
4. Set development goals

### Training Planning

1. Look for columns (criteria) with lots of red/yellow
2. These indicate skills the team needs to develop
3. Plan training sessions around common gaps

### Recognition

1. Find employees with mostly green scores
2. Recognize their excellence publicly
3. Ask them to help train others

## Classic View vs. Dashboard

| Feature | Classic View | Main Dashboard |
|---------|-------------|----------------|
| **Best for** | Printing, display | Analysis, navigation |
| **Layout** | Spreadsheet grid | Tables with expandable rows |
| **Interaction** | View only | Click to expand, filter, sort |
| **Detail level** | One position at a time | Multiple views available |

<Tip>
Use Classic View for quick visual overviews and printing. Use the main dashboard when you need to dig into details or track history.
</Tip>
