# Positions

Positions define the roles team members can work and be rated on. Each position has a name, description, and FOH/BOH assignment.

## Accessing Positions

1. Go to **Organization Settings**
2. Select **Positional Excellence** in the left menu
3. Click the **Positions** tab

## Understanding Positions

### What is a Position?

A position represents a specific job function in your restaurant:

- **iPOS** - Taking orders at the register
- **Host** - Greeting guests and managing the dining room
- **Breading** - Breading chicken products
- **Fries** - Managing waffle fry production

### FOH vs BOH

Positions are categorized as:

- **FOH (Front of House)** - Customer-facing roles
- **BOH (Back of House)** - Kitchen and production roles

Some positions may be marked as both if they span areas.

## Managing Positions

### Adding a Position

<Steps>
  <Step title="Click Add Position">
    Find the **Add Position** button below the existing positions.
  </Step>
  
  <Step title="Enter Position Name">
    Type the name of the position (e.g., "Expeditor", "Drive-Thru")
  </Step>
  
  <Step title="Set FOH/BOH">
    Check the appropriate boxes for where this position belongs.
  </Step>
  
  <Step title="Add Description">
    Write a description explaining what this position does. This appears when leaders submit ratings.
  </Step>
  
  <Step title="Save">
    Click away from the description field to auto-save, or click **Save Changes**.
  </Step>
</Steps>

### Editing a Position

1. Find the position in the list
2. Click on the name or description field to edit
3. Make your changes
4. Changes auto-save when you click away

### Removing a Position

1. Click the delete icon (trash) next to the position
2. Confirm the deletion

<Warning>
Deleting a position removes it from future ratings. Historical ratings for that position are preserved but the position won't appear in new rating submissions.
</Warning>

## Default Positions

When you first set up Levelset, common restaurant positions are included:

**FOH Positions:**
- iPOS
- Host
- Drinks
- Bagging
- Runner
- OMD (Order Management)

**BOH Positions:**
- Primary
- Secondary
- Machines
- Breading
- Prep
- Fries

You can customize these or add new ones as needed.

## Positions and Rating Criteria

Each position needs rating criteria defined in the **Rating Criteria** tab. After adding a position:

1. Go to the **Rating Criteria** tab
2. Select the new position from the dropdown
3. Define the 5 rating criteria

See [Rating Criteria](/organization-settings/rating-criteria) for details.

## Positions and Role Mapping

Control which roles can rate each position in the **Role Mapping** tab:

1. Go to the **Role Mapping** tab
2. Find the position in the list
3. Select which roles can submit ratings for that position

See [Roles](/organization-settings/roles) for more on role configuration.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Position not appearing in rating form">
    Check that:
    - The position has rating criteria defined
    - The user's role has permission to rate that position
    - The position is marked as active
  </Accordion>
  
  <Accordion title="Can't delete a position">
    Positions with historical ratings cannot be fully deleted. Contact support if you need to hide a position.
  </Accordion>
  
  <Accordion title="Description not saving">
    Ensure you click outside the text field to trigger the auto-save. If issues persist, try clicking the Save Changes button explicitly.
  </Accordion>
</AccordionGroup>
