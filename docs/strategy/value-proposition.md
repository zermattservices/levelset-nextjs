## The "Connected" Differentiator

What makes Levelset fundamentally different is that **every feature feeds into every other feature:**

```
RATING submitted for Team Member on iPOS position
    → Updates rolling average for that position
    → Updates OE Pillar scores (if criteria map to pillars)
    → May trigger Pending certification status
        → Enables formal Evaluation
            → Passing evaluation leads to Certified status
                → Certification unlocks higher pay rate
                → Development Plan milestone completed
                    → Pathway progress updated
    → Levi AI can instantly surface this data
    → Leaders see impact in Team Overview
    → Position Rankings update automatically
```

```
INFRACTION documented for Team Member
    → Points added to 90-day rolling total
    → If threshold crossed → Recommended Discipline Action generated
    → Leader takes action → documented with timestamp
    → Team Member acknowledges → recorded
    → History feeds into Evaluation consideration
    → Development Plan may include discipline-related milestones
    → Levi AI surfaces discipline context in coaching conversations
    → Retention risk indicators update
```

**This connectivity is what paper, spreadsheets, and disconnected tools can never provide.**

## Why Chick-fil-A Only?

1. **Language matters.** CFA has specific terminology (Operator, Team Member, iPOS, Drive-Thru, FOH/BOH) that generic software ignores. Levelset speaks CFA.

2. **Culture matters.** CFA's focus on development, care, and operational excellence is unique. Levelset is built around this culture, not adapted to it.

3. **Operations matter.** The OE Pillars (Great Food, Quick & Accurate, Creating Moments, Caring Interactions, Inviting Atmosphere) are CFA's framework. Levelset measures against them natively.

4. **Focus matters.** By serving one market exclusively, every feature, every workflow, every screen is designed for CFA operations. No compromises, no "configure it yourself" generic tools.

5. **Community matters.** CFA Operators talk to each other. A platform that earns trust in the CFA community grows through word of mouth. Being CFA-exclusive builds that trust faster.
